Send Custom Message
With 3 API and Twilio

This project demonstrates my proficiency in web development and API integration through the utilization of Twilio, a versatile cloud communications platform. By incorporating three API calls and Twilio's messaging feature, this webpage facilitates seamless message sending for various purposes, such as marketing campaigns or event notifications, offering a cost-effective and efficient solution.

The webpage comprises three main pages:

Home: This page collects user details, then redirects to the success page along with the user parameters.

Sucess: Using the provided user information, this page sends a message via Twilio. Upon successful transmission, the page renders with a gif of check mark symbol. If there's an error, the user is redirected to the error page. error.html: This page displays error messages encountered during the process.

Error: This page displays error messages encountered during the process.

Note: In case of an invalid city entry, the default city used is "Sudbury"."
